Bot Settings

Welcome Message

When enabled, a welcome message will be sent when members join the server.

New Member Role

When enabled, new members will be given a role when they join.

Leave Message

When enabled, a leave message will be sent when members leave the server.

Message Functions

When enabled, the specified actions will take place when a message containing the trigger is set.

Set actions:

Starboard [WIP]

When enabled, messages that meet the reaction requiremnets will be posted in the starboard channel.

Modmail [WIP]

When enabled, members will be abled to contact moderators via DMs with the bot.

Bot Activity Status

When enabled, the bot will display or cycle through the specified status.
